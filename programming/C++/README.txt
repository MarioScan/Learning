This is the repository with all the exercises and exam practices I made in 2025 
to pass the exam of Programmation 2 in C++.

Contains:
- Fundamentals of OOP (classes, objects and derivated classes)
- Data Structures: linked lists, queues, stacks, trees, BSTs (Binary Search Trees), queues with priority
- Fundamental algorithms based on the Big(O) complexity: Selection Sort, Insertion Sort, Bubble Sort, Merge Sort, Quick Sort
- Overload of Functions
- Templates for generic functions.
