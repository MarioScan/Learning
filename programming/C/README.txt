This is the repository with all the exercises and exam practices I made in 2024
to pass the exam of Programmation 1 in C.

Contains:
- Use of common libraries
- Building own local libraries
- Structs, enum, unions
- Reading and writing from/on a file. 
- Data Structures: linked lists, queues, stacks
- Simple algorithms: Selection Sort, Insertion Sort, Bubble Sort
- Recursion
- Sequencial research and binary search
